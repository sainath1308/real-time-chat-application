# 💬 Real-Time Chat Application

A real-time messaging application enabling live communication between users with instant message delivery.

## 🚀 Features

- Real-time bidirectional messaging
- User authentication and session management
- Chat rooms / direct messaging support
- Live online/offline user status
- Clean and intuitive chat UI

## ⚙️ Setup & Run

1. Clone the repo
```bash
   git clone https://github.com/sainath1308/real-time-chat-application.git
```
2. Install dependencies:
```bash
   pip install -r requirements.txt
```
3. Run the app:
```bash
   python app.py
```
4. Open `http://localhost:5000` in your browser
